迭代式 Prompt 开发（Iterative Prompt Development）

Prompt 开发是下列三个步骤的重复：

  1. 想出 idea。
  2. 设计实验，包括收集代码和数据，设计 Prompt。
  3. 根据实验结果进行错误分析，refine idea 和 Prompt。
