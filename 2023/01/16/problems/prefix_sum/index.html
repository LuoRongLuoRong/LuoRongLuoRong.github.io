前缀和

下面给出两个例子。

  • 第一个是给定 target，计算和为 target 的子数组的数目；
  • 第二个是给定 target，计算和为 target 的子数组的最短的长度。

解题的核心思想都是一致的：

  1. 使用 Map 记录出现过的 prefix_sum 的数目/最近下标。
  2. 处理特殊值，即将 0 放入 Map 中。
  3. 进入循环：
    1. 查找 Map 中是否存在 prefix_sum - target 的值，更新 res。
    2. 把本轮循环的 prefix_sum 放入 Map 中。
  4. 退出循环，返回 res。
